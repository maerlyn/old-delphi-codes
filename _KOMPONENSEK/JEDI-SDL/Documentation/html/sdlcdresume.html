SDL_CDResume

Name

SDL_CDResume -- Resumes a CDROM

Synopsis

uses
  SDL;

function SDL_CDResume( cdrom : PSDL_CD ) : Integer;

Description

Resumes play on the given cdrom.

Return Value

Returns 0 on success, or -1 on an error.

See Also

SDL_CDPlay, SDL_CDPause
