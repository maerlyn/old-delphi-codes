SDL_CDPause

Name

SDL_CDPause -- Pauses a CDROM

Synopsis

uses
  SDL;

function SDL_CDPause( cdrom : PSDL_CD ) : Integer;

Description

Pauses play on the given cdrom.

Return Value

Returns 0 on success, or -1 on an error.

See Also

SDL_CDPlay, SDL_CDResume
