SDL_WaitEvent

Name

SDL_WaitEvent -- Waits indefinitely for the next available event.

Synopsis

uses
  SDL;

function SDL_WaitEvent( event : PSDL_Event ) : Integer;

Description

Waits indefinitely for the next available event, returning 1, or 0 if there was an error while waiting for events.

If event is not nil, the next  event is removed from the queue and stored in that area.
