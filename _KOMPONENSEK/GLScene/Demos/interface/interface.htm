Interface demos :

  • camera :
    • moving camera with the mouse (rotation & translation)
    • zooming/unzooming with the mouse wheel
  • pick :
    • finding which object(s) was picked with the mouse in a GLSceneViewer
    • coloring objects under the mouse pointer
  • fadingintf :
    • smoother, progressive coloring of objects under the pointer
    • using the OnProgress event
  • bmpfont :
    • using a THUDText and TBitmapFont to render 2D text
    • rotating and scaling a 2D text
  • cursor :
    • using a THUDSprite as mouse cursor, with optional alpha-blended trail
    • loading and displaying a picture, retrieving the color under the mouse
